# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-09-26

### Added

-   Initial release of the Advanced Calculator project.
-   **Web Application**: Flask-based front-end with a responsive UI.
-   **GUI Application**: Tkinter-based desktop front-end.
-   **CLI Application**: Command-line interface for terminal-based calculations.
-   **Core Logic**: Central `logic.py` module for all mathematical operations.
-   **Testing**: Unit tests for the core logic module.
-   **Documentation**:
    -   `README.md` with project overview and setup instructions.
    -   `DOCUMENTATION.md` with detailed technical architecture and API reference.
    -   `CHANGELOG.md` to track project versions.
